Built-in Pipes

Angular comes with several built-in pipes for common transformations.

Data Formatting Pipes

These are used to format data like strings, dates, and numbers.

Uppercase

Description: Converts text to uppercase.

Example:

{{ 'hello' | uppercase }} → {{ 'hello' | uppercase }}

Lowercase

Description: Converts text to lowercase.

Example:

{{ 'HELLO' | lowercase }} → hello

Titlecase

Description: Capitalizes the first letter of each word.

Example:

{{ 'angular pipes' | titlecase }} → Angular Pipes

Date

Description: Formats dates based on predefined or custom formats.

Example:

{{ today | date:'shortDate' }} → 24/12/2024

Currency

Description: Formats numbers as currency.

Example:

{{ 1234 | currency:'USD' }} → $1,234.00

Decimal

Description: Formats numbers with decimal points.

Example:

{{ 1234.567 | number:'1.2-2' }} → 1,234.57

Percent

Description: Converts numbers to percentages.

Example:

{{ 0.25 | percent }} → 25%
